# Harness reference

theta-spec casts to the harnesses listed below. Each page documents the casting surface, known limitations, and harness-specific behavior.

Harness metadata is tracked in [`harnesses.toml`](https://github.com/tamarillo-ai/theta-spec/blob/main/harnesses.toml). If a page is stale, [file an issue](https://github.com/tamarillo-ai/theta-spec/issues/new?labels=stale-harness).

|---|---|---|
